# What I Use

With the time I have used many different technologies to work. This is a list of my current stack.

## Tools

* [Visual Studio Code](https://code.visualstudio.com/) as my main IDE.
* [Google Chrome](https://chrome.google.com) as my main browser.
* [Prettier](https://prettier.io) (with a VSCode plugin) with the default configuration.
* [Hyper](https://hyper.is/) as my terminal.
* [Postman](https://www.getpostman.com/) to try API endpoints
* [GitHub](https://github.com) for my code repositories (including private ones).

## Tech Stack

* [JavaScript](https://www.javascript.com/) as my main programming language.
* [React.js](https://reactjs.org) for UI creation.
* [Next.js](https://github.com/zeit/next.js) as my Frontend framework.
* [Now.sh](https://now.sh) for deploying applications to the cloud.
* [Micro](https://github.com/zeit/micro) for creating microservices.
* [GraphQL](http://graphql.org/) for API creations.
* [TravisCI](http://travis-ci.org/) for continuous integration pipelines.
